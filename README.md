2位数相加，结果是3-4位。
-->m位+n位 = 结果位数[m+n-1,m+n]


很大的2个数实现相加：读取2个数，当字符串处理，生成结果字符数组，长度是较大数 的位数或位数+1；
将这两个数 拆成字符数组，各位对齐，相加，逢十进一，得到逆序结果字符。
